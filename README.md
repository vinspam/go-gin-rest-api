# __go-gin-rest-api__

Golang gin framework REST API with JWT authentication

__URL Endpoints:__

``http://localhost:8080``

| Endpoint  | Method |
| ------------- | ------------- |
| ``/login?email=info@aironax.com&password=test``  | __POST__  |
| ``/emi``  | __POST__ |

## Follow the steps to Run App

    go mod tidy
    
    go mod vendor
    
    go mod run main.go
